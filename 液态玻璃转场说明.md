# 液态玻璃转场效果 - 功能说明

## ✨ 新增功能

已为青棠API主页添加**液态玻璃转场效果**，当用户点击导航按钮时，会出现优雅的液态玻璃扩散动画。

## 🎯 触发场景

### 导航栏按钮
- **首页** → 平滑滚动到顶部
- **说明** → 液态玻璃转场到"说明"模块
- **入口模块** → 液态玻璃转场到"入口模块"

### 页面内链接
- **查看三大入口** 按钮
- **向下滚动** 箭头按钮
- 所有带 `href="#xxx"` 的锚点链接

## 🎨 视觉效果

1. **点击触发**：在点击位置生成一个液态玻璃气泡
2. **扩散动画**：气泡从小到大扩散，带有：
   - 毛玻璃模糊效果 (backdrop-filter blur)
   - 柔和的阴影
   - 透明度渐变
   - 平滑的缩放动画
3. **页面跳转**：气泡扩散到一定程度时，平滑滚动到目标位置
4. **自动清理**：动画完成后自动清除元素

## 🔧 技术实现

### CSS 特性
```css
.liquid-blob {
  backdrop-filter: saturate(180%) blur(40px);
  animation: liquid-expand 0.8s cubic-bezier(0.22, 1, 0.36, 1);
}
```

### 动画时间轴
- **0-400ms**：液态玻璃气泡扩散动画
- **400ms**：触发平滑滚动到目标位置
- **400-700ms**：淡出转场覆盖层
- **700ms**：清理DOM元素

### 适配性
- ✅ 亮色/暗色主题自适应
- ✅ 响应式设计（桌面/平板/手机）
- ✅ 尊重用户的 `prefers-reduced-motion` 设置
- ✅ 外部链接不受影响（正常打开）

## 🐛 故障排查

### 问题1：点击后没有转场效果
**可能原因：**
- 浏览器不支持 `backdrop-filter`
- 系统启用了"减少动画"选项

**解决方法：**
1. 使用现代浏览器（Chrome/Edge/Safari）
2. 检查系统辅助功能设置

### 问题2：转场效果卡顿
**可能原因：**
- 设备性能较低
- 浏览器硬件加速未启用

**解决方法：**
1. 在浏览器设置中启用硬件加速
2. 关闭其他占用资源的程序

### 问题3：外部链接也触发转场
**检查：**
- 外部链接应该有 `target="_blank"` 属性
- JavaScript 只处理 `href` 以 `#` 开头的链接

## 🎨 自定义配置

如果你想调整转场效果，可以修改以下参数：

### 调整动画时长
```css
/* 在 CSS 中找到 */
.liquid-blob {
  animation: liquid-expand 0.8s /* 改为 1s 或 0.6s */
}
```

### 调整气泡大小
```javascript
// 在 JavaScript 中找到
blob.style.width = '100px';  // 改为 150px 或 80px
blob.style.height = '100px'; // 改为 150px 或 80px
```

### 调整模糊程度
```css
/* 在 CSS 中找到 */
backdrop-filter: saturate(180%) blur(40px); /* 改为 blur(20px) 或 blur(60px) */
```

### 调整转场延迟
```javascript
// 在 JavaScript 中找到
setTimeout(() => {
  targetElement.scrollIntoView(...);
}, 400); // 改为 300 或 500
```

## 💡 设计理念

液态玻璃转场效果的设计灵感来自：
- **iOS/macOS** 的模糊效果和流动动画
- **Material Design** 的涟漪效果
- **Glassmorphism** 设计趋势

目标是创造一种**柔和、现代、高端**的用户体验，同时：
- 不分散注意力
- 提供视觉反馈
- 增强品牌感知
